# Setup

This is how you install the core library:

# Install With NPM

npm i @box2d/core

# Install With Yarn

yarn add @box2d/core

You can install the other packages in exactly the same way:

  • @box2d/lights
  • @box2d/particles
  • @box2d/controllers
