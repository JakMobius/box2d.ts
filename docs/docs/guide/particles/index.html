# Particles

This is a TypeScript port of the particles code found in LiquidFun (opens new window) by Google. For now, you'll have to read the official documentation (opens new window) and adjust it to the TypeScript version. Make sure to check out the examples (opens new window) as well.

We try to stay true to the original API where we can, but sometimes it's not possible. We'll cover the differences in upcoming tutorials.

Make sure to check out the Setup guide first!
